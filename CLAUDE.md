# English Conversation AI App - Agent Summary

## アプリ概要

- AIエージェントを介して英会話を行う学習アプリ。
- 左側に会話表示ウィンドウ、右側にユーザーサポートウィンドウを配置する2ペインUI。
- ユーザーが発話した英文をAIに渡し、AIの返答を表示する形式で対話を行う。
- サポート画面では、例文提案や日本語でのQ&Aを提供し、学習を助ける。

## 主な機能

### 会話ウィンドウ
- ユーザー英文の入力と送信を行う。
- AIの返答を受け取り、チャット形式で表示する。

### サポートウィンドウ
- 例文表示やユーザー質問への回答を担当。
- モード切替や支援機能を提供する。

### Examモード
- Exam セッションを新規作成するとチャレンジ形式の学習が開始される。
- 相手の発話音声だけが聞こえるモードを想定している。
- AIエージェントがランダムな話題を提示する。
- ユーザーは10ラリーの対話を行う。
- 対話後に評価が行われ、点数とフィードバックが表示される。

### ペルソナ設定
- AIエージェントの人格をユーザーがプロンプトで設定できる。
- 応答スタイルやキャラクターをカスタマイズ可能。

## 目的

- 英語学習者がAIと自然な英会話の練習をできるようにする。
- 会話だけでなく、例文や日本語サポートで学習効果を高める。
- Examモードで自分の英語力を試すチャレンジ機能を提供する。
- AIの性格を変更して、対話の幅や学習体験を柔軟にする。

## 開発要件

### 技術スタック

- Next.js
- TypeScript
- React
- Tailwind CSS
- Next.js API Routes
- 音声入力 / 出力サービス（`services/stt.ts` / `services/tts.ts`）

### コミュニケーション
- 日本語で応答する（コード・変数名は英語）
- 簡潔に回答し、自明な説明は省略する
- 複雑なタスクでは実装前に計画を提示し、承認後に着手する

### コードスタイル
- 関数型アプローチを優先し、副作用を最小化する
- 厳密な型付け（anyは使わずunknownを使う）
- エラーは握りつぶさず、意味のあるメッセージ付きで処理する

### Git規約
- Conventional Commits形式、本文は日本語（例: `feat: ユーザー認証にOAuth2を追加`）
- 確認なしに自動コミット・自動pushしない

- テストコードを確認なしに削除・コメントアウトしない
